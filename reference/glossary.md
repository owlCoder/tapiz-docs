# Glossary

| Term | Meaning |
|---|---|
| Assistant | Teaching staff role that manages subjects and academic workflows |
| Student | End user who scans attendance QR codes and views academic data |
| Faculty Manager | Faculty-scoped administrative role |
| Super Admin | Platform-wide administrative role |
| Subject | Course/module entity |
| Session | Scheduled teaching event with QR attendance |
| Score Sheet | Grade/points matrix with rows, columns and cells |
| Sheet Version | Snapshot/diff history for a score sheet |
| Public Material Token | Share token for material access without normal subject context |
| Incident | Status-page event shown through the status endpoints |
